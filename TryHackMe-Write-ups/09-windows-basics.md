## 2. Objective

The objective of this room is to understand the basics of the Microsoft Windows operating system, including its interface, file system, user management, and built-in administrative tools commonly used in IT and cybersecurity.

---

## 3. Introduction

Microsoft Windows is one of the most widely used operating systems in personal and enterprise environments. It provides a graphical user interface (GUI), supports multitasking, manages hardware resources, and includes various tools for system administration and security.

For cybersecurity professionals, understanding Windows is essential because it is a common target for cyber attacks and is widely used in organizations.

---

## 4. Key Concepts

### Desktop Environment

The Windows desktop is the primary workspace where users access applications, files, and system settings.

Main components include:

- Desktop
- Start Menu
- Taskbar
- Notification Area

---

### File Explorer

File Explorer is the built-in file management tool used to browse drives, folders, and files.

Common operations include:

- Create folders
- Copy files
- Move files
- Rename files
- Delete files
- Search for files

---

### User Accounts

Windows allows multiple user accounts with different permission levels.

Common account types include:

- Administrator
- Standard User
- Guest (legacy/disabled in most modern versions)

---

### File System

Windows primarily uses the **NTFS (New Technology File System)**.

Key features of NTFS include:

- File permissions
- Encryption support
- Compression
- Large file support
- Access Control Lists (ACLs)

---

### Control Panel & Settings

These tools allow users to configure system settings, manage hardware, install software, and customize the operating system.

---

### Task Manager

Task Manager is used to monitor and manage running applications, background processes, CPU usage, memory usage, disk activity, and network utilization.

It can also be used to end unresponsive applications.

---

### Windows Security

Windows includes several built-in security features:

- Microsoft Defender Antivirus
- Windows Firewall
- User Account Control (UAC)
- Windows Update
- BitLocker (supported editions)

These features help protect the system against malware and unauthorized access.
