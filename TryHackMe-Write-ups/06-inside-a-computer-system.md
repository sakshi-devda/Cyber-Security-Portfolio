## 2. Objective

The objective of this room is to understand the basic hardware components of a computer system, how they work together, and why they are important in cybersecurity.

---

## 3. Introduction

A computer system is made up of multiple hardware components that work together to process data, execute programs, store information, and communicate with other devices. Understanding these components is essential for troubleshooting, system administration, and cybersecurity.

---

## 4. Key Concepts

### Central Processing Unit (CPU)

The CPU is the brain of the computer. It executes instructions, performs calculations, and controls the overall operation of the system.

---

### Random Access Memory (RAM)

RAM is temporary memory used to store data and programs currently in use. It provides fast access for the CPU but loses its contents when the computer is powered off.

---

### Storage Devices

Storage devices permanently store the operating system, applications, and user data.

Examples:

- Hard Disk Drive (HDD)
- Solid State Drive (SSD)

---

### Motherboard

The motherboard is the main circuit board that connects all hardware components, allowing them to communicate with each other.

---

### Input Devices

Input devices allow users to provide data to the computer.

Examples:

- Keyboard
- Mouse
- Scanner
- Microphone

---

### Output Devices

Output devices display or present processed information.

Examples:

- Monitor
- Printer
- Speakers

---

### Power Supply Unit (PSU)

The Power Supply Unit converts electrical power from the wall outlet into the voltages required by the computer's internal components.

## 7. Key Takeaways

- Every hardware component has a specific role in system operation.
- The CPU processes instructions and coordinates system activities.
- RAM provides temporary storage for active processes.
- Storage devices retain data permanently.
- Understanding hardware basics is important for cybersecurity and troubleshooting.
