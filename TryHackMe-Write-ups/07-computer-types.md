## 2. Objective

The objective of this room is to understand the different types of computers, their characteristics, and how they are used in various environments such as personal computing, business, cloud computing, and enterprise infrastructure.

---

## 3. Introduction

Computers are designed to perform different tasks depending on their purpose. Some are built for personal use, while others are designed to manage large-scale business operations, scientific research, or cloud services. Understanding these computer types helps cybersecurity professionals identify potential security risks and protect different computing environments.

---

## 4. Key Concepts

### Personal Computer (PC)

A Personal Computer is designed for individual users and is commonly used for web browsing, office work, programming, gaming, and education.

Examples:

- Desktop Computer
- Laptop

---

### Workstation

A Workstation is a high-performance computer built for professional tasks such as software development, video editing, 3D modeling, engineering, and scientific research.

Compared to a regular PC, workstations typically offer:

- Faster processors
- More RAM
- Professional-grade graphics
- Higher reliability

---

### Server

A Server is a computer that provides services, applications, files, or resources to other computers (clients) over a network.

Common server types include:

- Web Server
- File Server
- Database Server
- Mail Server
- Application Server

---

### Mainframe

A Mainframe is a powerful computer designed to process massive amounts of data and support thousands of users simultaneously.

Mainframes are commonly used by:

- Banks
- Government organizations
- Insurance companies
- Large enterprises

---

### Supercomputer

A Supercomputer is the fastest and most powerful type of computer, used for complex calculations and scientific simulations.

Common uses include:

- Weather forecasting
- Artificial Intelligence
- Space research
- Scientific simulations
- Medical research

---

### Mobile Devices

Mobile devices are portable computing systems designed for communication and everyday tasks.

Examples:

- Smartphones
- Tablets
- Smartwatches

These devices require strong security measures due to their portability and constant internet connectivity.

---

### Virtual Machines (VMs)

A Virtual Machine is a software-based computer that runs inside another physical computer using virtualization technology.

Virtual machines are widely used for:

- Testing software
- Malware analysis
- Penetration testing
- Running multiple operating systems
- Cybersecurity labs
