# Bash Basics in Linux

## 📌 What is Bash?

Bash (Bourne Again Shell) is a command-line interpreter used in Linux to interact with the operating system. It allows users to execute commands, run scripts, and automate tasks.

---

## 🎯 Why is Bash Important?

- Execute Linux commands efficiently
- Automate repetitive tasks using scripts
- System administration and monitoring
- Essential for cybersecurity and penetration testing
- Used in incident response automation

---

## 🧠 What is a Shell?

A shell is a program that acts as an interface between the user and the operating system.

Bash is the most commonly used shell in Linux.

---

## 🖥️ Types of Shells

- Bash (most common)
- sh (Bourne shell)
- zsh (Z shell)
- ksh (Korn shell)

---

## 📋 Basic Bash Concepts

### 1. Command Execution

```bash
ls
pwd
whoami
```

Each command is executed by the shell.

---

### 2. Variables in Bash

```bash
name="Linux"
echo $name
```

Variables store values that can be reused.

---

### 3. User Input

```bash
read user_input
echo "You entered: $user_input"
```

Used to take input from users.

---

### 4. Echo Command

```bash
echo "Hello World"
```

Used to display output.

---

### 5. Comments in Bash

```bash
# This is a comment
echo "Hello"
```

Comments are ignored by the system.

---

## 🔁 Basic Operators

### 1. Logical Operators

- `&&` → AND
- `||` → OR

Example:

```bash
mkdir test && cd test
```

---

### 2. Command Chaining

```bash
ls; pwd; whoami
```

Runs multiple commands sequentially.

## ⭐ Best Practices

- Always validate user input in scripts
- Avoid running unknown scripts
- Use comments for readability
- Keep scripts simple and modular
- Test scripts before production use
