# Linux Networking Commands Practical

## 🎯 Objective

Learn how to use basic Linux networking commands to check network configuration, test connectivity, and troubleshoot network issues.

---

## 🧪 Lab Environment

- Operating System: Kali Linux
- Virtual Machine: VirtualBox
- Terminal: Bash

# 🖥️ Practical 1: Display Network Interfaces

## Command

```bash
ip addr
```

## Purpose

Displays all network interfaces along with their IP addresses.

## Screenshot

## Explanation

The `ip addr` command provides detailed information about all available network interfaces, including IPv4 and IPv6 addresses.

# 🖥️ Practical 4: Test Network Connectivity

## Command

```bash
ping google.com
```

## Purpose

Checks whether the system can communicate with another host.

## Screenshot

> 📸 Screenshot will be added later.

## Explanation

The `ping` command sends ICMP Echo Request packets to verify network connectivity and measure response time.

> **Note:** Press **Ctrl + C** to stop the ping process.
