# Packet Capturing Practical

## 🎯 Objective

Learn how to capture live network packets using Wireshark, generate network traffic, stop the capture, and save the captured packets for future analysis.

---

# Prerequisites

Before starting this practical, ensure that:

- Wireshark is installed.
- Npcap is installed.
- You have Administrator privileges.
- Your system is connected to the Internet.

---

# Step 1: Launch Wireshark

Open **Wireshark** from the Start Menu or Desktop shortcut.

The home screen will display all available network interfaces.

---

# Step 2: Select a Network Interface

Choose the interface that is currently connected to the Internet.

Examples:

- Wi-Fi
- Ethernet

Double-click the interface to begin capturing packets.

---

# Step 3: Verify Packet Capture

Once packet capture starts, packets should begin appearing continuously in the Packet List Pane.

If packets are visible, Wireshark is successfully capturing network traffic.

---

# Step 4: Generate Network Traffic

To capture meaningful packets, perform a few network activities.

You can:

- Open Google
- Open YouTube
- Refresh any website
- Run a Ping command

Example:

```bash
ping google.com
```

You should notice new packets appearing in real time.

---

# Step 5: Stop Packet Capture

After capturing sufficient packets:

- Click the **Red Stop** button on the toolbar.

The packet capture will stop, and all captured packets will remain available for analysis.

---

# Step 6: Save the Capture File

Navigate to:

**File → Save As**

Save the file using the `.pcapng` format.

Example:

```
packet-capture.pcapng
```

---

# Expected Result

After completing this practical, you should be able to:

- Capture live network traffic.
- Generate packets by performing network activities.
- Stop packet capturing.
- Save captured packets in `.pcapng` format.
