# Display Filters Practical

## 🎯 Objective

Learn how to use Display Filters in Wireshark to analyze specific network traffic efficiently.

---

# Prerequisites

Before starting this practical, ensure that:

- Wireshark is installed.
- A packet capture has been completed.
- You have a capture file (.pcapng) or live network traffic.

---

# What is a Display Filter?

A Display Filter allows you to view only the packets that match a specific condition.

It does **not** remove or modify any packets from the capture.

---

# Practical 1: Display TCP Packets

### Filter

```text
tcp
```

### Steps

1. Open a packet capture.
2. Click on the Display Filter bar.
3. Type:

```text
tcp
```

4. Press **Enter**.

### Expected Result

Only TCP packets should be displayed.

# Practical 5: Display ICMP Packets

```text

```

```bash

```


# Observation

- Display Filters make packet analysis much easier by showing only relevant traffic.
- Different filters help identify specific protocols, IP addresses, and ports.
- Applying the correct filter reduces analysis time and improves troubleshooting.

# Conclusion

In this practical, various Display Filters were applied to analyze different types of network traffic. These filters help security analysts and network administrators quickly isolate relevant packets for troubleshooting, monitoring, and cybersecurity investigations.
