# Wireshark Statistics - Theory

## 1. Introduction

Wireshark Statistics is a feature that helps in analyzing captured network traffic by providing summarized information about packets, protocols, endpoints, conversations, and network behavior.

It helps security analysts understand:
- Network communication patterns
- Traffic volume
- Protocol usage
- Suspicious activities
- Performance issues


## 2. Why Statistics are Important

Statistics help to quickly analyze large packet captures without manually checking every packet.

Uses:
- Finding top communicating devices
- Identifying unusual traffic
- Understanding protocol distribution
- Troubleshooting network problems
- Supporting security investigations


## 3. Statistics Menu in Wireshark

Wireshark provides multiple statistical analysis options:

- Summary
- Protocol Hierarchy
- Conversations
- Endpoints
- I/O Graphs
- Service Response Time
- Flow Graph


## 4. Summary Statistics

Summary provides an overview of the capture file.

It shows:
- Capture duration
- Total packets
- Average packet rate
- File size
- Data transferred

It helps in getting a quick understanding of the captured traffic.


## 5. Protocol Hierarchy

Protocol Hierarchy displays protocols present in the capture.

It shows:
- Protocol name
- Number of packets
- Percentage of traffic
- Data size

Example:
- TCP
- UDP
- HTTP
- DNS
- TLS

Security analysts use it to identify unexpected protocols.


## 6. Endpoints

Endpoints show individual network devices involved in communication.

It provides information about:

- IP addresses
- MAC addresses
- Packet count
- Bytes transferred

Example:
- Client IP
- Server IP


## 7. Conversations

Conversations show communication between two endpoints.

It displays:

- Source address
- Destination address
- Protocol
- Packets exchanged
- Bytes transferred

Useful for identifying major communication sessions.


## 8. I/O Graphs

I/O Graphs provide a visual representation of network traffic over time.

Uses:
- Detecting traffic spikes
- Monitoring bandwidth usage
- Finding unusual activity patterns


## 9. Flow Graph

Flow Graph shows packet communication sequence between hosts.

It helps understand:
- Client-server communication
- Request and response flow
- Connection behavior


## 10. Security Uses of Statistics

Wireshark statistics are useful for:

- Detecting port scanning
- Finding suspicious connections
- Identifying malware communication
- Investigating network attacks
- Analyzing abnormal traffic


## 11. Conclusion

Wireshark Statistics provides a fast way to analyze network captures.

Features like Protocol Hierarchy, Conversations, Endpoints, and I/O Graphs help security professionals understand network behavior and detect suspicious activities.
