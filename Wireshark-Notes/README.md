# Wireshark Notes

Welcome to my **Wireshark Notes** repository.

This repository contains structured Wireshark notes created as part of my Cyber Security learning journey. The notes focus on packet capturing, packet analysis, protocol inspection, and network troubleshooting using Wireshark. These concepts are essential for ethical hacking, penetration testing, SOC analysis, digital forensics, and incident response.

## 📚 Topics Covered

* TCP Three-Way Handshake
* HTTP Packet Analysis
* DNS Packet Analysis
* ICMP Packet Analysis
* ARP Protocol
* TCP vs UDP
* Wireshark Filters
* Follow TCP Stream
* Export Objects
* Packet Analysis Challenges

## 🎯 Objectives

* Learn how to capture and analyze network traffic.
* Understand how common network protocols work.
* Develop packet analysis and troubleshooting skills.
* Prepare for cybersecurity interviews and practical assessments.
* Build hands-on experience with Wireshark.
