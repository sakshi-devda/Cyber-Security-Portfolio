# Wireshark Installation & Interface Practical

## 🎯 Objective

Learn how to install Wireshark, understand its interface, and perform the first packet capture.

---

# Prerequisites

- Windows 10/11 (or Linux/macOS)
- Administrator privileges
- Stable Internet Connection

---

# Step 1: Download Wireshark

1. Open your browser.
2. Visit the official Wireshark website.

https://www.wireshark.org/

3. Click **Download**.
4. Download the latest stable version for your operating system.

---

# Step 2: Install Wireshark

1. Run the downloaded installer.
2. Click **Next** until the component selection screen appears.
3. Keep the default components selected.
4. Continue the installation.

---

# Step 3: Install Npcap

During installation, Wireshark will ask to install **Npcap**.

Npcap is required because Wireshark cannot capture network packets without a packet capture driver.

Recommended options:

- ✔ Install Npcap
- ✔ Install Npcap in WinPcap API-compatible Mode

Click **Install**.

---

# Step 4: Launch Wireshark

After installation,

- Open Wireshark.
- The home screen will display all available network interfaces.

Examples:

- Wi-Fi
- Ethernet
- Loopback Adapter
- Virtual Network Adapters (if installed)

---

# Understanding the Home Screen

The home screen contains:

- Menu Bar
- Toolbar
- Available Network Interfaces
- Capture Filter Bar
- Recent Capture Files

---

# Step 5: Select a Network Interface

Choose the interface currently connected to the Internet.

For most users:

- Wi-Fi
- Ethernet

Double-click the interface to start capturing packets.

---

# Step 6: Generate Some Network Traffic

While Wireshark is capturing packets, perform a few activities:

- Open Google
- Open YouTube
- Refresh a webpage
- Run a Ping command

Example:

```
ping google.com
```

Wireshark will immediately begin displaying captured packets.

---

# Step 7: Stop Packet Capture

Click the **Red Stop Button** on the toolbar.

The captured packets will remain available for analysis.

---

# Basic Interface Overview

Wireshark consists of three main sections.

## 1. Packet List Pane

Displays all captured packets.

Information shown includes:

- Packet Number
- Time
- Source IP
- Destination IP
- Protocol
- Length
- Info

---

## 2. Packet Details Pane

Displays detailed protocol information for the selected packet.

Example:

- Ethernet
- IP
- TCP
- HTTP

Each protocol layer can be expanded for deeper analysis.

---

## 3. Packet Bytes Pane

Displays the raw packet data in hexadecimal and ASCII format.

Useful during forensic investigations and malware analysis.

---

# Saving a Capture

To save the captured packets:

File → Save As

Choose:

- .pcapng (Recommended)
- .pcap

---

# Expected Result

After completing this practical, you should be able to:

- Install Wireshark successfully.
- Understand the main interface.
- Capture live packets.
- Stop packet capture.
- Save the capture file.
