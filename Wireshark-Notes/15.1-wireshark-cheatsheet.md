# Wireshark Cheatsheet

## 1. Introduction

This cheatsheet contains important Wireshark display filters, capture filters, shortcuts, and analysis techniques used in network traffic analysis.


# 2. Basic Display Filters

## Show All Traffic

Filter:
all

Shows all packets in the capture.


## Filter by Protocol

TCP:
tcp

UDP:
udp

HTTP:
http

HTTPS/TLS:
tls

DNS:
dns


# 3. IP Address Filters

## Specific IP Address

Filter:
ip.addr == 192.168.1.10

Shows all traffic related to a specific IP address.


## Source IP

Filter:
ip.src == 192.168.1.10


## Destination IP

Filter:
ip.dst == 192.168.1.10



# 4. Port Filters

## Specific Port

Filter:
tcp.port == 443

Common ports:

80   - HTTP
443  - HTTPS
53   - DNS


## Source Port

Filter:
tcp.srcport == 443


## Destination Port

Filter:
tcp.dstport == 80



# 5. TCP Analysis Filters

## TCP Traffic

Filter:
tcp


## TCP SYN Packets

Filter:
tcp.flags.syn == 1


## TCP Retransmissions

Filter:
tcp.analysis.retransmission


## Follow TCP Stream

Steps:

Right click packet

Follow → TCP Stream


Used to view complete TCP communication.



# 6. DNS Analysis Filters

## DNS Queries

Filter:
dns.flags.response == 0


## DNS Responses

Filter:
dns.flags.response == 1


## Search Specific Domain

Filter:
dns.qry.name contains "example"



# 7. HTTP Analysis Filters

## HTTP Requests

Filter:
http.request


## HTTP Response

Filter:
http.response


## HTTP GET Requests

Filter:
http.request.method == "GET"


## HTTP POST Requests

Filter:
http.request.method == "POST"



# 8. TLS / HTTPS Analysis Filters

## TLS Traffic

Filter:
tls


## TLS Handshake

Filter:
tls.handshake


## Client Hello

Filter:
tls.handshake.type == 1


## Server Hello

Filter:
tls.handshake.type == 2



# 9. Wireshark Statistics

## Summary

Location:

Statistics → Capture File Properties

Provides:

- Capture duration
- Packet count
- File size


## Protocol Hierarchy

Location:

Statistics → Protocol Hierarchy

Shows:

- Protocol usage
- Packet percentage
- Traffic distribution


## Endpoints

Location:

Statistics → Endpoints

Shows:

- Active hosts
- IP addresses
- Packet count


## Conversations

Location:

Statistics → Conversations

Shows:

- Source and destination communication
- Data transfer details


## I/O Graph

Location:

Statistics → I/O Graph

Used for:

- Traffic visualization
- Detecting traffic spikes



# 10. Export Objects

Location:

File → Export Objects

Used for:

- Extracting transferred files
- Malware investigation
- File recovery


Supported protocols:

- HTTP
- SMB
- TFTP

# 14. Security Analysis Checklist

Always check:

- Unknown IP addresses
- Suspicious domains
- Unusual ports
- Large data transfers
- DNS anomalies
- Malicious file downloads
- TCP errors
