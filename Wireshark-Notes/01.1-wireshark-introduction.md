# Wireshark Introduction

## 📌 Objective

Understand what Wireshark is, why it is used, its features, and its importance in networking and cybersecurity.

---

# What is Wireshark?

Wireshark is a free and open-source network protocol analyzer used to capture, inspect, and analyze network traffic in real time.

It allows users to see every packet traveling through a network interface and helps understand how devices communicate over a network.

Wireshark is one of the most widely used tools by:

- Network Administrators
- Cyber Security Analysts
- Penetration Testers
- SOC Analysts
- Incident Responders
- Students learning Computer Networks

---

# Why is Wireshark Used?

Wireshark helps analyze network communication by capturing packets exchanged between devices.

It is commonly used to:

- Troubleshoot network issues
- Monitor network traffic
- Detect suspicious activities
- Analyze malware communication
- Investigate cyber attacks
- Learn networking protocols
- Debug applications using network communication

---

# What is a Packet?

A packet is a small unit of data transmitted over a network.

Whenever you:

- Open a website
- Send an email
- Watch YouTube
- Ping another computer
- Download a file

your device sends and receives thousands of packets.

Wireshark captures these packets for analysis.

---

# How Does Wireshark Work?

The basic workflow of Wireshark is:

1. Select a network interface.
2. Start packet capture.
3. Capture live network traffic.
4. Stop capturing.
5. Analyze captured packets.
6. Apply filters to locate specific traffic.

---

# Features of Wireshark

- Live Packet Capture
- Deep Packet Inspection
- Display Filters
- Capture Filters
- Colorized Packet View
- Protocol Analysis
- Export Captured Packets
- Follow TCP Stream
- Statistics and Graphs
- Supports hundreds of network protocols

# Wireshark in Cyber Security

Wireshark is an essential tool for cybersecurity professionals.

It helps to:

- Detect malicious network traffic
- Analyze phishing attacks
- Investigate malware communication
- Detect data exfiltration
- Identify unauthorized devices
- Analyze suspicious DNS requests
- Investigate brute-force attacks
- Monitor network anomalies

---

# Advantages

- Free and Open Source
- Beginner Friendly
- Powerful Packet Analysis
- Cross Platform
- Supports Multiple Protocols
- Widely Used in Industry

---

# Limitations

- Cannot decrypt encrypted traffic without keys.
- Requires packet capture permissions.
- Large captures can become difficult to analyze.
- Does not actively block attacks.
