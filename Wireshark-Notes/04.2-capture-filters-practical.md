# Capture Filters Practical

## 🎯 Objective

Learn how to use Capture Filters in Wireshark to capture only specific network traffic and reduce unnecessary packets during packet capture.

---

# Prerequisites

Before starting this practical, ensure that:

- Wireshark is installed.
- Npcap is installed.
- Your system is connected to the Internet.
- You have Administrator privileges.

---

# What is a Capture Filter?

A Capture Filter is applied **before** packet capturing begins. It instructs Wireshark to capture only packets that match the specified condition.

Unlike Display Filters, packets that do not match the filter are **not captured**.

---

# Practical 1: Capture Only TCP Packets

### Capture Filter

```text
tcp
```

### Steps

1. Open Wireshark.
2. Select your active network interface.
3. Enter the filter in the **Capture Filter** field.
4. Start packet capture.

### Expected Result

Only TCP packets should be captured.
