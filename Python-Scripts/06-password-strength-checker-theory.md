## 2. Objective

The objective of this project is to create a Python program that checks the strength of a password based on common security standards. The program helps users create stronger passwords and understand password security best practices.

---

## 3. Introduction

Passwords play a vital role in protecting online accounts and sensitive information. Weak passwords are vulnerable to brute-force and dictionary attacks.

A Password Strength Checker evaluates a password using criteria such as length, uppercase letters, lowercase letters, numbers, and special characters. It then classifies the password as Weak, Moderate, or Strong.

---

## 4. Python Module Used

### re

The `re` (Regular Expressions) module is used to search for specific patterns within a password.

It helps identify:

- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

---

## 5. Working Principle

The program works as follows:

1. The user enters a password.
2. The program checks the password length.
3. It verifies the presence of uppercase letters.
4. It checks for lowercase letters.
5. It searches for numeric digits.
6. It checks for special characters.
7. Based on the results, the password is classified as Weak, Moderate, or Strong.
