## 2. Objective

The objective of this project is to create a Python program that scans a range of hosts on a network and determines whether they are active or inactive. This project introduces basic network discovery techniques using Python.

---

## 3. Introduction

A Network Scanner is a tool used to discover devices connected to a network. It helps identify active hosts, troubleshoot network connectivity, and understand network topology.

Network scanning is one of the first steps performed during network reconnaissance and is commonly used by system administrators and cybersecurity professionals.

**Note:** This project should only be used on systems and networks where you have permission to perform scanning.

---

## 4. Python Modules Used

### socket

The `socket` module is used to establish network connections and verify whether a host is reachable.

### ipaddress

The `ipaddress` module helps generate and validate IP addresses within a network.

---

## 5. Working Principle

The program works as follows:

1. The user enters a network address.
2. The program generates IP addresses within the specified range.
3. Each IP address is checked for connectivity.
4. Active hosts are displayed.
5. After scanning is complete, a summary message is shown.
