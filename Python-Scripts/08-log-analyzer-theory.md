## 2. Objective

The objective of this project is to create a Python program that reads a log file and analyzes its contents. The program helps identify important log entries such as errors, warnings, and informational messages.

---

## 3. Introduction

Log analysis is an essential task in cybersecurity and system administration. Logs record events that occur within operating systems, applications, and network devices.

By analyzing logs, security professionals can detect suspicious activities, troubleshoot problems, monitor system health, and investigate security incidents.

---

## 4. Python Modules Used

### os

The `os` module is used to access files and directories.

### collections

The `collections` module helps count different types of log entries efficiently.

---

## 5. Working Principle

The program works as follows:

1. The user specifies a log file.
2. The program opens the log file.
3. Each line is read and analyzed.
4. Error, Warning, and Information messages are counted.
5. A summary report is displayed.

---

## 6. Features

- Reads log files
- Counts Error messages
- Counts Warning messages
- Counts Information messages
- Displays analysis summary
- Beginner-friendly implementation

## 8. How to Run

1. Open the project in Visual Studio Code.
