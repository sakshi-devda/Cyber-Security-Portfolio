## 2. Objective

The objective of this project is to create a Python program that generates strong and random passwords. The generated passwords help improve account security by reducing the risk of password guessing and brute-force attacks.

---

## 3. Introduction

Passwords are the first line of defense for protecting online accounts and sensitive information. Weak passwords can be easily guessed or cracked using automated tools.

A password generator creates random passwords using uppercase letters, lowercase letters, numbers, and special characters, making them much stronger and more secure than manually created passwords.

---

## 4. Python Modules Used

### random

The `random` module is used to randomly select characters while generating the password.

### string

The `string` module provides predefined character sets such as:

- Uppercase Letters (A–Z)
- Lowercase Letters (a–z)
- Digits (0–9)
- Special Characters (!, @, #, $, %, etc.)

---

## 5. Working Principle

The program works as follows:

1. The user enters the desired password length.
2. The program combines letters, numbers, and symbols into one character set.
3. Random characters are selected from the character set.
4. The selected characters are combined to create the password.
5. The generated password is displayed to the user.
