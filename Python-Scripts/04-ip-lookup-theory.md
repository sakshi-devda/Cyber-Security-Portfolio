## 2. Objective

The objective of this project is to create a Python program that retrieves basic information about an IP address, such as its hostname. This project helps understand how IP addresses can be identified and resolved using Python.

---

## 3. Introduction

Every device connected to a network has an Internet Protocol (IP) address. An IP Lookup tool helps identify information related to an IP address or hostname.

In this project, Python's built-in **socket** module is used to resolve hostnames into IP addresses and perform basic lookups. This introduces the fundamentals of DNS resolution and network communication.

---

## 4. Python Module Used

### socket

The `socket` module provides networking capabilities in Python.

It is used to:

- Resolve hostnames
- Retrieve IP addresses
- Perform basic network communication

---

## 5. Working Principle

The program works as follows:

1. The user enters a domain name or hostname.
2. The program sends a DNS lookup request.
3. The hostname is resolved into its corresponding IP address.
4. The IP address is displayed to the user.
