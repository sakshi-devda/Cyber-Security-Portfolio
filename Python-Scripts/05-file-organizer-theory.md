## 2. Objective

The objective of this project is to create a Python program that automatically organizes files into different folders based on their file extensions. This project introduces file handling and automation using Python.

---

## 3. Introduction

Managing a large number of files manually can be time-consuming. A File Organizer automates this process by sorting files into separate folders according to their file types.

For example:

- Images → Images Folder
- Documents → Documents Folder
- Videos → Videos Folder
- Audio → Audio Folder

Automation scripts like this improve productivity and are widely used by system administrators and cybersecurity professionals.

---

## 4. Python Modules Used

### os

The `os` module is used to:

- Access files and folders
- Create directories
- Rename files
- Move files
- Read folder contents

---

### shutil

The `shutil` module is used for high-level file operations such as:

- Moving files
- Copying files
- Managing directories

---

## 5. Working Principle

The program works as follows:

1. Read all files from a selected folder.
2. Check the extension of each file.
3. Create folders if they do not already exist.
4. Move files into their respective folders.
5. Display a success message after organizing all files.

## 8. How to Run

1. Open the project in Visual Studio Code.
2. Open the terminal.
3. Run the following command:

```bash
python 05-file-organizer.py
```

4. Enter the folder path (or use the predefined folder).
5. The program will organize the files automatically.
