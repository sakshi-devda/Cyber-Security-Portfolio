# Wireless Networking

## 📌 What is Wireless Networking?

Wireless Networking is a method of connecting devices to a network without using physical cables. Instead of Ethernet cables, devices communicate using radio waves.

Wireless networks provide flexibility, mobility, and easy access to network resources.

---

## 🎯 Why is Wireless Networking Important?

Wireless networking helps to:

- Connect devices without cables.
- Support mobility.
- Simplify network deployment.
- Reduce installation costs.
- Provide Internet access in homes, offices, schools, and public places.

---

# 🌐 How Wireless Networking Works

A wireless network typically consists of:

- Wireless Devices (Laptop, Smartphone, Tablet)
- Wireless Access Point (AP) or Wi-Fi Router
- Internet Service Provider (ISP)

Communication occurs through radio frequency (RF) signals between the device and the access point.

---

# 📡 Wi-Fi Standards

Wi-Fi standards are defined by the IEEE 802.11 family.

| Standard | Frequency | Features |
|----------|-----------|----------|
| 802.11a | 5 GHz | Higher speed, shorter range |
| 802.11b | 2.4 GHz | Longer range, lower speed |
| 802.11g | 2.4 GHz | Improved speed |
| 802.11n (Wi-Fi 4) | 2.4/5 GHz | Better range and performance |
| 802.11ac (Wi-Fi 5) | 5 GHz | High-speed wireless networking |
| 802.11ax (Wi-Fi 6) | 2.4/5 GHz | Higher efficiency, better performance in crowded environments |

# 🔒 Wireless Security Protocols

## WEP (Wired Equivalent Privacy)

- Very old security protocol.
- Weak encryption.
- Easily broken.
- **Not recommended.**

---

## WPA (Wi-Fi Protected Access)

- Improved security compared to WEP.
- Better encryption.
- Mostly replaced by WPA2 and WPA3.

---

## WPA2

- Strong encryption using AES.
- Most commonly used wireless security protocol.
- Suitable for home and business networks.

---

## WPA3

- Latest wireless security standard.
- Improved protection against password attacks.
- Stronger authentication and encryption.

# ⚠️ Common Wireless Security Threats

Wireless networks are vulnerable to attacks such as:

- Rogue Access Points
- Evil Twin Attacks
- Packet Sniffing
- Deauthentication Attacks
- Weak Wi-Fi Passwords
- Unauthorized Access

---

# 🛡️ Wireless Networking in Cybersecurity

Cybersecurity professionals secure wireless networks by:

- Using WPA3 or WPA2 encryption.
- Disabling outdated protocols like WEP.
- Monitoring connected devices.
- Detecting rogue access points.
- Performing wireless security assessments.
- Implementing strong authentication.

---

## 💼 Real-World Example

A company deploys WPA3-protected Wi-Fi for employees.

Guest users connect to a separate guest network that has restricted Internet-only access, reducing the risk of unauthorized access to internal resources.

---

## ⭐ Best Practices

- Use WPA3 whenever possible.
- Use strong Wi-Fi passwords.
- Disable WEP.
- Change default router credentials.
- Keep router firmware updated.
- Separate guest and internal wireless networks.
- Monitor connected devices regularly.
