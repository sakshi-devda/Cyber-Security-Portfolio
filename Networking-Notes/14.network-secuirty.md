# Network Security

## 📌 What is Network Security?

Network Security is the practice of protecting computer networks, devices, and data from unauthorized access, cyber attacks, misuse, and data breaches.

It combines technologies, policies, and security controls to ensure that network resources remain secure, available, and reliable.

---

## 🎯 Why is Network Security Important?

Network security helps to:

- Prevent unauthorized access.
- Protect sensitive information.
- Detect cyber attacks.
- Maintain data confidentiality.
- Ensure business continuity.
- Reduce security risks.

Without proper network security, organizations become vulnerable to cyber threats and data loss.

---

# 🛡️ Goals of Network Security

The primary goals of network security are:

### 1. Confidentiality

Ensures that only authorized users can access sensitive information.

Example:

Encrypted communication using HTTPS.

---

### 2. Integrity

Ensures that data is not modified without authorization.

Example:

Hashing and digital signatures.

---

### 3. Availability

Ensures that systems and services remain accessible when needed.

Example:

Protection against Denial-of-Service (DoS) attacks.

These three principles are collectively known as the **CIA Triad**.

---

# 🔒 Common Network Security Controls

## Firewall

Filters incoming and outgoing network traffic.

---

## Intrusion Detection System (IDS)

Monitors network traffic and generates alerts when suspicious activity is detected.

---

## Intrusion Prevention System (IPS)

Monitors network traffic and automatically blocks malicious activity.

---

## Virtual Private Network (VPN)

Encrypts communication over public networks.

---

## Antivirus and Endpoint Protection

Detects and removes malware from endpoints.

---

## Multi-Factor Authentication (MFA)

Adds an extra layer of security beyond passwords.

---

# ⚠️ Common Network Threats

Organizations face many network threats, including:

- Malware
- Phishing
- Ransomware
- Denial-of-Service (DoS)
- Man-in-the-Middle (MITM)
- Password attacks
- Insider threats

---

# 📋 Best Practices for Network Security

- Use strong passwords.
- Enable Multi-Factor Authentication.
- Keep software updated.
- Use firewalls and IDS/IPS.
- Encrypt sensitive communication.
- Regularly back up important data.
- Monitor network logs.
- Disable unused services and ports.
- Follow the Principle of Least Privilege.

---

## 🛡️ Network Security in Cybersecurity

Cybersecurity professionals perform various tasks to secure networks, including:

- Monitoring network traffic.
- Detecting suspicious behavior.
- Responding to security incidents.
- Performing vulnerability assessments.
- Implementing security policies.
- Investigating cyber attacks.

Strong network security is one of the core responsibilities of a SOC Analyst.

---

## 💼 Real-World Example

A company deploys:

- Firewalls to filter traffic.
- IDS/IPS to detect and prevent attacks.
- VPN for remote employees.
- MFA for employee logins.
- Endpoint protection on all computers.

Together, these controls significantly reduce the organization's attack surface.

---

## ⭐ Best Practices

- Follow the principle of least privilege.
- Keep operating systems and applications updated.
- Regularly review firewall rules.
- Monitor network traffic continuously.
- Conduct regular security assessments.
- Train employees to recognize cyber threats.
