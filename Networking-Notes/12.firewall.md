# Firewalls

## 📌 What is a Firewall?

A Firewall is a network security device or software that monitors, filters, and controls incoming and outgoing network traffic based on predefined security rules.

It acts as a barrier between a trusted internal network and an untrusted external network, such as the Internet.

---

## 🎯 Why is a Firewall Important?

A firewall helps to:

- Prevent unauthorized access.
- Block malicious traffic.
- Protect sensitive data.
- Monitor network connections.
- Enforce security policies.

It is one of the first lines of defense in network security.

---

# 🌐 How Does a Firewall Work?

When data enters or leaves a network:

1. The firewall examines the packet.
2. It compares the packet with configured security rules.
3. If the traffic matches an allowed rule, it is permitted.
4. If it violates a rule, it is blocked or rejected.

This process helps protect systems from unauthorized or malicious traffic.

---

# 🔥 Types of Firewalls

## 1. Packet Filtering Firewall

Examines individual packets based on:

- Source IP Address
- Destination IP Address
- Port Number
- Protocol

### Advantages

- Fast
- Simple
- Low resource usage

### Disadvantages

- Limited inspection
- Cannot analyze application data

---

## 2. Stateful Inspection Firewall

Tracks active network connections and makes decisions based on the connection state.

### Advantages

- More secure than packet filtering
- Better at detecting suspicious traffic

### Disadvantages

- Uses more system resources

---

## 3. Proxy Firewall

Acts as an intermediary between the client and the destination server.

### Advantages

- Hides internal systems
- Filters application traffic
- Improves security

### Disadvantages

- Slower due to additional processing

---

## 4. Next-Generation Firewall (NGFW)

A Next-Generation Firewall combines traditional firewall features with advanced security capabilities.

### Features

- Deep Packet Inspection (DPI)
- Intrusion Prevention System (IPS)
- Application awareness
- Malware detection
- SSL/TLS inspection

---

# 📊 Firewall Deployment Types

## Network Firewall

Protects an entire network by filtering traffic entering or leaving it.

Example:

A firewall installed between an organization's network and the Internet.

---

## Host-Based Firewall

Installed on an individual computer or server.

Examples:

- Windows Defender Firewall
- Linux UFW
- iptables
- firewalld

---

## 🛡️ Firewalls in Cybersecurity

Cybersecurity professionals use firewalls to:

- Block unauthorized access.
- Restrict unnecessary services.
- Prevent malware communication.
- Control network traffic.
- Detect suspicious connection attempts.

Firewalls are an essential part of a defense-in-depth security strategy.

---

## 💼 Real-World Example

A company hosts a public web server.

The firewall is configured to allow only:

- Port 80 (HTTP)
- Port 443 (HTTPS)

All other incoming ports are blocked, reducing the attack surface.

---

## ⭐ Best Practices

- Allow only required ports and services.
- Block unused ports.
- Regularly review firewall rules.
- Keep firewall software and firmware updated.
- Monitor firewall logs for suspicious activity.
- Follow the principle of least privilege.
