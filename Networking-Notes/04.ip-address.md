# IP Address

## 📌 What is an IP Address?

An IP (Internet Protocol) Address is a unique numerical identifier assigned to every device connected to a network. It allows devices to communicate and exchange data with each other.

Think of an IP address as the **home address** of a device. Just as a postal address helps deliver letters to the correct house, an IP address helps deliver data to the correct device.

---

## 🎯 Why is an IP Address Important?

- Identifies devices on a network.
- Enables communication between devices.
- Helps route data across networks.
- Essential for accessing the Internet.
- Used in network troubleshooting and cybersecurity.

---

# 🌐 Types of IP Addresses

## 1. IPv4

IPv4 is the most widely used version of the Internet Protocol.

### Features

- 32-bit address
- Four decimal numbers separated by dots
- Approximately 4.3 billion unique addresses

### Example

```text
192.168.1.10
```

---

## 2. IPv6

IPv6 was introduced to overcome the shortage of IPv4 addresses.

### Features

- 128-bit address
- Written in hexadecimal format
- Supports a much larger address space

### Example

```text
2001:0db8:85a3:0000:0000:8a2e:0370:7334
```

---

# 🌍 Public IP Address

A Public IP Address is assigned by an Internet Service Provider (ISP) and is accessible over the Internet.

Example:

```text
103.25.45.100
```

---

# 🏠 Private IP Address

A Private IP Address is used within a local network and cannot be accessed directly from the Internet.

### Common Private IP Ranges

| Range | Purpose |
|--------|---------|
| 10.0.0.0 – 10.255.255.255 | Large private networks |
| 172.16.0.0 – 172.31.255.255 | Medium private networks |
| 192.168.0.0 – 192.168.255.255 | Home and office networks |

---

# 🔄 Static vs Dynamic IP

## Static IP

- Does not change
- Manually configured
- Commonly used for servers

## Dynamic IP

- Assigned automatically by DHCP
- Can change over time
- Commonly used for personal devices

---

# 📡 How IP Addresses Work

When you visit a website:

1. Your device sends a request using its IP address.
2. Routers forward the request across networks.
3. The destination server receives the request.
4. The server sends the response back to your IP address.

---

## 🛡️ IP Addresses in Cybersecurity

Cybersecurity professionals use IP addresses to:

- Identify devices on a network.
- Detect suspicious activity.
- Block malicious IP addresses.
- Investigate cyber attacks.
- Analyze network logs.

Attackers may attempt to hide or spoof their IP addresses, making monitoring and analysis even more important.

---

## 💼 Real-World Example

A SOC Analyst notices repeated login attempts from an unfamiliar IP address. After reviewing logs, the IP is identified as malicious and is blocked through the organization's firewall.

---

## ⭐ Best Practices

- Do not expose sensitive systems directly to the Internet.
- Monitor network traffic for suspicious IP addresses.
- Keep firewall rules updated.
- Use private IP addresses within internal networks whenever possible.
