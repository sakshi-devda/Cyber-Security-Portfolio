# 📊 Packet Analysis Examples

## 1. TCP Three-Way Handshake

TCP establishes a connection using:

```
SYN
SYN-ACK
ACK
```

Explanation:

- SYN: Client requests a connection.
- SYN-ACK: Server responds and accepts the request.
- ACK: Client confirms the connection.

This process creates a reliable communication channel.

---

## 2. DNS Query Analysis

DNS converts domain names into IP addresses.

Example flow:

```
Client
  |
  ↓
DNS Request
  |
  ↓
DNS Server
  |
  ↓
DNS Response
```

Analyzed information:

- Domain name
- Source IP
- Destination IP
- Query response

---

## 3. HTTP Packet Analysis

HTTP traffic analysis helps understand web communication.

Analyzed:

- HTTP Requests
- HTTP Responses
- Headers
- Data Transfer

## 2. Applying Filters

Common Wireshark filters used:

### TCP Filter

```
tcp
```

Displays TCP packets.

---

### DNS Filter

```
dns
```

Displays DNS queries and responses.

---

### HTTP Filter

```
http
```

Displays HTTP communication.

---

### ICMP Filter

```
icmp
```

Displays ICMP packets such as ping requests.
