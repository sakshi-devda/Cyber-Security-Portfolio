# Network Scanner Project

## 1. Project Overview

The Network Scanner Project is a Python-based cybersecurity tool designed to identify active devices within a network. It scans a specified range of IP addresses and determines which hosts are reachable.

This project demonstrates the fundamentals of network discovery and basic reconnaissance techniques used in cybersecurity.

---

## 2. Objective

The objective of this project is to develop a simple network scanner using Python that can identify active hosts within an authorized network.

---

## 3. Problem Statement

Network administrators and cybersecurity professionals need to identify devices connected to a network for monitoring, troubleshooting, and security assessments.

Manually checking each device is inefficient, so an automated network scanner helps simplify this process.

## 6. Project Workflow

2. The program generates IP addresses.
3. Each IP address is checked.
4. Active hosts are identified.
5. Results are displayed.
