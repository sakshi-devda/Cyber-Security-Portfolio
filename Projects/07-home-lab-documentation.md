# 🖥️ Home Lab Documentation

## 📌 Project Overview

Home Lab Documentation is a cybersecurity project that explains the setup and configuration of a personal cybersecurity practice environment.

A home lab provides a safe environment to practice security tools, perform experiments, and develop practical cybersecurity skills without affecting real systems.

---

## 🎯 Objective

The main objectives of this project are:

- Build a cybersecurity practice environment
- Understand virtual lab setup
- Practice security tools safely
- Perform hands-on experiments
- Improve Linux and networking skills

---

## 🛠️ Tools & Technologies Used

- VirtualBox / VMware
- Kali Linux
- Linux Operating System
- Windows Virtual Machine
- Networking Concepts
- Cybersecurity Tools

---

## ⚙️ Lab Components

The home lab includes:

- Attacker Machine:
  - Kali Linux

- Target Machines:
  - Windows/Linux Virtual Machines

- Security Tools:
  - Nmap
  - Wireshark
  - Metasploit Framework
  - Burp Suite

---

## 🧠 How It Works

The home lab setup follows these steps:

1. Install virtualization software.
2. Create virtual machines.
3. Configure network settings.
4. Install cybersecurity tools.
5. Perform security testing in an isolated environment.
6. Document findings and learning outcomes.

## ▶️ Lab Setup Process

### Step 1: Install Virtualization Software

Used:

- VirtualBox
- VMware Workstation

Purpose:

To create and manage virtual machines.

---

### Step 2: Install Kali Linux

Kali Linux is configured as the primary security testing environment.

Used for:

- Network scanning
- Security testing
- Tool practice

---

### Step 3: Configure Network

Network modes:

- NAT
- Host-only Adapter
- Internal Network

Purpose:

To allow communication between lab machines safely.

---

### Step 4: Install Security Tools

Installed tools:

```bash
sudo apt update
```

Examples:

```bash
nmap
wireshark
burpsuite
metasploit-framework
```
