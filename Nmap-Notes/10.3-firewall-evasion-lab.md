# Firewall Evasion Basics Lab

## 🎯 Objective

In this lab, you will evaluate how different Nmap scan options affect firewall behavior in an **authorized lab environment**. You will compare scan results, identify filtering behavior, and document your findings.

---

# Lab Environment

- Operating System: Kali Linux
- Tool: Nmap
- Target: TryHackMe Machine / Hack The Box Machine / Your Own Virtual Machine

---

# Task 1: Baseline Scan

## Command

```bash
nmap <target-ip>
```

### Observe

- Host status
- Open ports
- Closed ports
- Filtered ports
- Scan duration

📸 **Screenshot:** Baseline scan

---

# Task 2: Fragmentation Test

## Command

```bash
sudo nmap -f <target-ip>
```

### Observe

- Compare results with the baseline scan.
- Check whether any port states changed.
- Record scan duration.

📸 **Screenshot:** Fragmentation scan

---

# Task 3: Decoy Scan

## Command

```bash
sudo nmap -D RND:5 <target-ip>
```

### Observe

- Compare detected ports.
- Check for any response differences.
- Record scan duration.

📸 **Screenshot:** Decoy scan

---

# Task 4: Source Port Test

## Command

```bash
sudo nmap --source-port 53 <target-ip>
```

### Observe

- Compare firewall responses.
- Record any changes in filtered ports.

📸 **Screenshot:** Source port scan

---

# Task 5: MAC Address Spoofing (Local Network)

## Command

```bash
sudo nmap --spoof-mac 00:11:22:33:44:55 <target-ip>
```

### Observe

- Compare results with previous scans.
- Note any differences.
