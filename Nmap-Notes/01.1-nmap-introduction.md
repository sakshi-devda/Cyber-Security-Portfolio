# Nmap Introduction

## 📌 What is Nmap?

Nmap (Network Mapper) is a free and open-source network scanning tool used for network discovery, security auditing, and vulnerability assessment.

It helps identify active hosts, open ports, running services, operating systems, and network configurations.

Nmap is one of the most widely used tools by cybersecurity professionals, ethical hackers, penetration testers, system administrators, and network engineers.

---

## 🎯 Why is Nmap Important?

Nmap helps security professionals to:

- Discover live hosts on a network.
- Identify open and closed ports.
- Detect running services.
- Determine service versions.
- Identify operating systems.
- Audit network security.
- Perform reconnaissance during penetration testing.

---

# 🌐 History of Nmap

- Developed by **Gordon Lyon**, also known as **Fyodor**.
- First released in **1997**.
- Written primarily in **C, C++, Lua, and Python**.
- Available for Linux, Windows, and macOS.
- Maintained as an open-source project.

---

# ⚙️ How Does Nmap Work?

Nmap sends specially crafted packets to target systems and analyzes their responses.

Based on these responses, it can determine:

- Whether a host is online.
- Which ports are open.
- Which services are running.
- Service versions.
- Operating system details.
- Firewall behavior.

# 📊 Advantages

- Free and open source.
- Fast and reliable.
- Supports multiple operating systems.
- Highly customizable.
- Large scripting engine (NSE).
- Excellent documentation and community support.

---

# ⚠️ Limitations

- Can be detected by IDS/IPS systems.
- Requires proper permissions for scanning.
- Some scan types require administrative/root privileges.
- Firewall rules may affect scan results.

# 💼 Real-World Example

A security analyst joins a new organization and needs to identify all active devices on the internal network.

Using Nmap, the analyst can:

- Discover active hosts.
- Identify open ports.
- Detect running services.
- Determine operating systems.
- Create an inventory of network assets.

---

## ⭐ Best Practices

- Obtain proper authorization before scanning.
- Scan only systems you are permitted to test.
- Use less aggressive scans in production environments.
- Document scan results.
- Regularly update Nmap to the latest version.
