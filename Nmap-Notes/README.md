# Nmap

## 📖 Overview

Nmap (Network Mapper) is one of the most popular open-source tools used for network discovery, port scanning, service detection, operating system detection, and security auditing.

It helps cybersecurity professionals understand the structure of a network, identify active hosts, discover open ports, detect running services, and find potential security risks.

Nmap is widely used by:

- Penetration Testers
- SOC Analysts
- Security Engineers
- Network Administrators
- Ethical Hackers
- Incident Response Teams
