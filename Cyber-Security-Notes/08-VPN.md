# VPN (Virtual Private Network)

## 📌 Definition
A VPN (Virtual Private Network) creates a secure, encrypted connection over a less secure network (like the Internet) to protect user privacy and data.

---

## 🎯 Purpose of VPN
- Hide user IP address
- Encrypt internet traffic
- Secure data on public networks
- Access restricted content safely

---

## ⚙️ How VPN Works
1. User connects to VPN server
2. VPN encrypts internet traffic
3. Data is sent through secure tunnel
4. Website sees VPN server IP, not real IP

---

## 🔐 Key Features
- Encryption of data
- IP masking
- Secure tunneling
- Bypassing geo-restrictions

---

## 🧠 Real World Example
Using public WiFi in a café:
- Without VPN → data can be intercepted
- With VPN → data is encrypted and safe

---

## 🧱 Types of VPN

### 1. Remote Access VPN
Used by individual users to connect securely to a network.

### 2. Site-to-Site VPN
Used by organizations to connect multiple office networks.

---

## 🛡️ Advantages
- Privacy protection
- Secure communication
- Safe public WiFi usage

---

## ⚠️ Limitations
- Can slow internet speed
- Not 100% anonymous
- Some VPNs are untrusted

---

## 🎯 Key Takeaway
VPN ensures secure and private communication over the internet by creating an encrypted tunnel.
