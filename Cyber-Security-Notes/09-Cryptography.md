# Cryptography in Cyber Security

## 📌 Definition
Cryptography is the technique of securing information by converting it into unreadable format (cipher text) so that only authorized users can access it.

---

## 🎯 Purpose of Cryptography
- Ensure data confidentiality
- Protect data integrity
- Authenticate users
- Prevent unauthorized access

---

## 🔐 Types of Cryptography

### 1. Symmetric Encryption
Same key is used for encryption and decryption.

**Example:**
- AES (Advanced Encryption Standard)
- DES (Data Encryption Standard)

---

### 2. Asymmetric Encryption
Uses two keys:
- Public Key (encryption)
- Private Key (decryption)

**Example:**
- RSA Algorithm

---

## ⚙️ How Cryptography Works
1. Plain text is converted to cipher text using encryption algorithm
2. Cipher text is transmitted over network
3. Receiver decrypts it using key
4. Data becomes readable again

---

## 🧠 Real World Example
When you use HTTPS websites:
- Data is encrypted using cryptography
- Prevents hackers from reading sensitive data

---

## 🔑 Key Concepts

### Plain Text
Original readable data

### Cipher Text
Encrypted unreadable data

### Encryption
Process of converting plain text to cipher text

### Decryption
Process of converting cipher text back to plain text

---

## 🛡️ Advantages
- Strong data protection
- Secure communication
- Authentication support

---

## ⚠️ Limitations
- Complex implementation
- Key management issues
- Computational overhead

---

## 🎯 Key Takeaway
Cryptography is the backbone of secure communication in modern digital systems.
