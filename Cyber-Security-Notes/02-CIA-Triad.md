# CIA Triad (Confidentiality, Integrity, Availability)

## 📌 Definition
CIA Triad is the foundation of Cyber Security that ensures protection of data and systems through three principles.

---

## 🔐 Confidentiality
Ensures that data is only accessible to authorized users.

### Example:
- Password protection
- Encryption
- Access control

---

## ✏️ Integrity
Ensures that data is not altered or tampered with.

### Example:
- Hashing (SHA-256)
- Digital Signatures
- File integrity checks

---

## 🌐 Availability
Ensures that systems and data are available when needed.

### Example:
- Backup systems
- Load balancing
- DDoS protection

---

## 🧠 Real World Example
Banking system must ensure:
- Only user can access account (Confidentiality)
- Balance is not modified illegally (Integrity)
- Website is always accessible (Availability)

---

## 🎯 Key Takeaway
CIA Triad ensures security, trust, and reliability of any system.
